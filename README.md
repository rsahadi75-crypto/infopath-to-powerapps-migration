# infopath-to-powerapps-migration
⚠️ How to Use This Repository
These scripts were built for a specific SharePoint environment and are not intended to be run as-is. Instead, the recommended approach is to:

Upload these files to a new conversation in Claude AI (claude.ai)
Share a sample XML file from your own InfoPath document library
Provide your SharePoint list names, field names, and site URLs
Ask Claude to adapt the scripts to your specific environment

These files give Claude the full context of a working migration pattern so it can quickly build a customized solution for your forms.
